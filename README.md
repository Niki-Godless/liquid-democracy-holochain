# Liquid Democracy on Holochain

A peer-to-peer liquid democracy platform where you can delegate votes or vote directly.

## Features
- ✅ Create proposals across 7 domains (Air, Water, Food, Infrastructure, Health, Education, Cooperation)
- ✅ Delegate voting power to trusted representatives
- ✅ Vote directly on issues you care about
- ✅ Fully decentralized - no servers

## Status
- Core DNA: Working
- Multi-agent testing: In progress
- UI: In development

## Installation

1. Download [Holochain Launcher](https://github.com/holochain/launcher/releases)
2. Download `val_democracy.happ` from [Releases](https://github.com/Niki-Godless/liquid-democracy-holochain/releases)
3. Open Launcher → Install hApp → Select the .happ file

## Technical Stack
- Rust + Holochain HDK
- Pure peer-to-peer architecture
- DHT-based storage
